# Universe Wall Framework

這是一個原創世界觀設定檔案庫，主要記錄「宇宙牆壁」、「虛」、「平行世界」、「平行鏈」、「有限宇宙」與「跨宇宙現象」等設定。

This is an original worldbuilding archive that mainly documents concepts such as Universe Walls, the Void, Parallel Worlds, Parallel Chains, Finite Universes, and cross-universe phenomena.

## 理念 Philosophy

本資料庫作為開放式世界觀框架公開。

This project is released as an open worldbuilding framework.

授權： CC BY 4.0
附加相容規則： Universe Wall Framework 的既有底層規則可以被擴展，但不能被覆寫。
任何人都可以基於本框架創作自己的小說、角色、世界、遊戲、漫畫、動畫與商業作品。
他們創作出的原創作品屬於他們自己。
但是，如果一部作品想宣稱自己「相容於官方 Universe Wall Framework」，就不能矛盾、刪除或重新定義已經建立的底層規則。
可以新增規則、世界、種族、能力系統、歷史與解釋，只要它們是在既有基礎上擴展，而不是取代基礎。

License: CC BY 4.0
Additional Compatibility Rule: The original base rules of the Universe Wall Framework may be expanded, but not overwritten.
People may create their own original stories, characters, worlds, games, comics, animations, and commercial works based on this framework.
Their original works belong to them.
However, to claim compatibility with the official Universe Wall Framework, a work must not contradict, erase, or redefine the established base rules.
New rules, worlds, species, systems, abilities, histories, and interpretations may be added, as long as they are built on top of the existing foundation rather than replacing it.
Works that change the base rules may still be published, but they should be labeled as “inspired by the Universe Wall Framework,” not “compatible with the Universe Wall Framework.”
